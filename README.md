# immoc-node
### Trying to learn to build a node commander line tool.

### npm package
- commander 命令行工具
- inquirer 命令行交互
```sh
# 此版本支持commonjs
npm install --save inquirer@^8.0.0
```
- download-git-repo 拉取远程仓库
```sh
npm i download-git-repo
```
- ora 加载动画
```sh
# 此版本支持commonjs
npm i ora@5
```
- chalk 终端样式
```sh   
npm i chalk@4
```
